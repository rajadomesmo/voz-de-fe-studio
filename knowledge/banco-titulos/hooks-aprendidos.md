# Banco de Hooks Aprendidos — Coletados de Vídeos Virais

> Preenchido pelo agent-researcher após cada ciclo de pesquisa.
> APPEND ONLY — nunca sobrescrever entradas anteriores.
> O agent-scriptwriter DEVE ler este arquivo antes de escrever qualquer hook.

---

## Como usar
1. Researcher coleta hooks reais de vídeos virais → classifica → gera 5 variações → salva aqui
2. Scriptwriter lê este arquivo → usa VARIAÇÕES dos padrões reais → nunca repete hook dos últimos 10 scripts
3. A cada ciclo o banco cresce — o agente fica mais preciso

## Regra de qualidade
Teste antes de usar: "esse hook para o scroll em 2 segundos?"
Se não → reescreva. Hook fraco = vídeo morto.

Hooks NUNCA usar:
- "Bem-vindo", "Olá", "Neste short de hoje"
- Qualquer abertura suave ou introdução antes da promessa
- Frases com mais de 12 palavras que não entregam urgência imediata
