## Insights para o Canal Voz de Fé

1. **Salmo 91 tem 50x mais potencial** que "declaração de fé" — focar produção aqui
2. **Thiago Agostinho é o benchmark absoluto** para salmo 91 — série diária, estética natureza + texto do salmo, narração grave
3. **Hook de identificação** ("você foi escolhido", "não é por acaso") converte bem mesmo com poucos seguidores
4. **Imagens rurais/natureza** ressoam fortemente com público feminino 30–55 anos evangélico
5. **"Oração poderosa" + dia da semana** ("Quarta-feira", "Segunda-feira") é formato replicável com baixo esforço
6. **Faceless + narração IA** é o formato padrão dos canais com mais views — alinha com nossa proposta
