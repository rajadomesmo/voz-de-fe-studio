# Referências Visuais

Prints de thumbnails e capturas de vídeos virais salvos pelo researcher.
Você pode fazer upload manual de prints aqui para análise.

Ao analisar um print, pergunte ao Claude Code:
"analise o print em knowledge\referencias-visuais\[nome-do-arquivo]"

## Convenção de nomes
`canal-tema-data.png`
Exemplos:
- oracoes-de-fe-protecao-2026-04-23.png
- soakstream-sono-2026-04-24.png

## O que salvar aqui
- Thumbnails de vídeos virais (Shorts e longos)
- Capturas de tela de resultados de busca (para análise de padrão visual dominante)
- Referências de estilo dark cinematográfico para prompts Gemini
- Prints de analytics de vídeos que performaram bem (se compartilhados pelo usuário)
