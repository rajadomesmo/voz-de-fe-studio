# Estrutura da pasta live\

## live\playlist\ — Conteúdo para Gyre.pro

Scripts de 30–60 min em loop contínuo.
Destinados à plataforma Gyre.pro para transmissão 24/7 no YouTube.
Formato: SLEEP PRAYER ou SALMO GUIADO com repetição intencional.
Produção: vídeo de fundo em loop (gerado no Gemini) + áudio ElevenLabs.

Arquivos: `oracao-loop-[N].md`
SEO correspondente: `seo\live-[N].md`

---

## scripts\ (pasta raiz do projeto) — Roteiros longos para YouTube

Scripts de 10–15 min para vídeos publicados normalmente no YouTube.
Não são conteúdo de live — são vídeos longos com estrutura completa:
abertura empática, âncoras bíblicas, oração principal, declarações finais, encerramento.

Arquivos: `scripts\roteiro-[N]-[tema].md`
SEO correspondente: `seo\long-[N].md`

---

## Resumo: as duas categorias são distintas

| | live\playlist\ | scripts\ |
|---|---|---|
| Duração | 30–60 min | 10–15 min |
| Destino | Gyre.pro (loop 24/7) | YouTube (vídeo normal) |
| Formato | Sleep prayer / loop | Roteiro completo |
| SEO | seo\live-[N].md | seo\long-[N].md |

Não há "live-01, 02, 03 faltando" — os roteiros em scripts\ são uma categoria separada.
